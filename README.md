Is project me child to parent prop pass karna seekh rahe hai. Child [Button.jsx] -> Parent [App.jsx] component me reverse props pass kr rahe hai or jo prop value hame [App.jsx] me mili hai usko [ColorSlate.jsx] me pass karke uska background color change kr rahe hai. 

How to reverse Props: Iske liye hum function ka use kr rahe hai. Parent component me ek funtion banayenge jo parameter lega (wahi parameter jo hame child component se chahiye as a prop), us parameter ko use krke function ka logic likhenge like state me save karke ya setState me update etc..., or us function ko child component me as a prop send krdenge. Child component me us function ko destructure karke ek dusre function ke ander usko call krdenge saath me parameter dedenge, wo parent component me jaake apna kaam krdega. Ab child component me alag function isiliye banaya tha kyouki hume kuch karna hoga to kuch onclick event jaise par karenge uske liye ek function banayenge, to wahi funtion ke ander hum utilize karenge prop wala function.
